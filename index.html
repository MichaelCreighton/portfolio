Recent Projects

Full stack weather app

This was my first "full stack" app. It's not particularly fancy to look at but there is a lot of new stuff (for me) going on in the background. There are two async api calls to two different apis, it uses node.js along with handlebars, and finally it is deployed to the web on Heroku.

A Simon Says copy

An old favourite from everyone's childhood! Click the button that the computer selects for you. Each round the computer will add one button to the pattern. Repeat the pattern for as long as you can. **Currently desktop/laptop only.**

Night at the Museum

Night at the museum is a lighthearted walk through a museum after dark, where some objects take on a life of their own as you enjoy the art. This was my first paired programming project, coded with my pal Carlie Dixon, while we were at JUNO college. It uses the API of the Chicago Art Institute to get all the art into your eyes.

Pics and Quotes

Pics and Quotes is sometimes a very deep and introspective experience and sometimes it's just hilarious. It uses two API calls to, first get a random quote from the Quote Garden API, and then a second one to get a random picture from Unsplash, matching the quote genre. With all this randomness, the connections are different for every user...

Bootcamp Diaries

Bootcamp Diaries is a minimalist app for writers, designed to provide an uncluttered page to work on. This was a group programming project while at the JUNO bootcamp, created with Corey, Taz, and Tiffany. It employs Firebase to store and retrieve writing prompts, for those with a touch of writers block. It also provides a pleasant light mode / dark mode toggle.

Under the Sea

Under the Sea is a multi-page web site for a large aquarium complex. This PSD conversion was made using a client brief and design files. Utilizing both CSS Flexbox and Grid it is fully responsive across all screen sizes. Crafted with pixel perfect detail, the clients design was re-created to exact specifications.
